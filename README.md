# QingYuan 简易库存管理系统

这是一个专为小型家具定制公司设计的轻量级库存管理系统。无需安装任何软件，直接在浏览器中运行。

## ✨ 主要功能

- **📊 仪表盘**：实时概览库存总量、分仓库库存、今日出入库数据及库存预警。
- **📦 商品管理**：轻松录入、编辑和管理商品信息，支持双仓库库存显示，型号和规格分开管理。
- **📥 入库管理**：支持选择“主仓库”或“销售门店”进行入库，自动增加对应库存。
- **📤 出库管理**：支持从不同位置出库，自动扣减对应库存。
- **🚚 库存调拨**：支持在“主仓库”和“销售门店”之间快速移动货物。
- **🖨️ 单据打印**：自动生成专业的入库单、出库单和调拨单。
- **🔍 历史查询**：详细记录每一笔流水（含位置信息），支持搜索和补打单据。
- **💾 数据安全**：数据保存在本地浏览器中，支持导出 Excel 表格（含双仓库数据）和 JSON 备份文件。

## 🚀 如何使用

1. **启动系统**：
   - 直接双击文件夹中的 `index.html` 文件即可打开系统。
   - 推荐使用 Chrome、Edge 或 Firefox 浏览器。

2. **初次使用建议**：
   - 先进入 **[商品管理]** 页面，录入您的基础商品信息。
   - 然后即可开始进行 **[入库]** 操作来初始化库存。

3. **数据备份**：
   - 建议定期（如每周）在 **[数据管理]** 页面点击 "导出数据 (Excel)" 或 "导出备份文件 (JSON)" 进行备份，防止电脑故障导致数据丢失。

## 📁 文件说明

- `index.html`: 系统主程序。
- `inventory.js`: 核心逻辑代码。
- `inventory.css`: 打印样式和辅助样式。
- `site_styles_backup.css`: 原有网站样式的备份文件（不影响本系统使用）。

## ⚠️ 注意事项

- **数据存储**：系统使用浏览器的 LocalStorage 技术，数据存储在您当前的浏览器中。如果您清除浏览器缓存，数据可能会丢失。请务必**定期导出备份**。
- **单机使用**：本系统设计为单机使用，不同电脑之间的数据不互通（除非手动导入/导出备份文件）。
